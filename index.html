Hero is a scion class skilled at engaging multiple enemies at once. Heroes can wield three types of weapons: Swords, Twin Machine Guns, and Talises.

Phantom is a scion class which features both a wide array of different attacks as well as superior dodging ability. The class is capable of equipping three different weapons types: Rod, Katana, and Rifle.

Etoile is a scion class specializing in defense and support skills. They are capable of equipping Double Sabers, Soaring Blades, and Wands.

Luster is an inherently agressive class with it's own unique mechanics, and a bit complicated and different from the other classes. It uses only Gunblades, but 3 different styles/types. (Original artwork can be found here)
